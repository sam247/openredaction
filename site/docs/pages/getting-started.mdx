# Getting Started

Get up and running with OpenRedaction in minutes.

## Installation

Install OpenRedaction via npm:

```bash
npm install openredaction
```

Or using yarn:

```bash
yarn add openredaction
```

Or using pnpm:

```bash
pnpm add openredaction
```

## Basic Usage

### Simple redaction

Create an `OpenRedaction` instance and call `detect()` (async):

```javascript
import { OpenRedaction } from 'openredaction';

const redactor = new OpenRedaction();
const text = "My name is John Doe and my email is john@example.com";
const result = await redactor.detect(text);

console.log(result.redacted);
// e.g. placeholders or masked text depending on redactionMode

console.log(result.detections);
// [{ type: 'NAME', value: 'John Doe', ... }, { type: 'EMAIL', value: 'john@example.com', ... }]
```

### With configuration

```javascript
import { OpenRedaction } from 'openredaction';

const redactor = new OpenRedaction({
  preset: 'gdpr',
  redactionMode: 'placeholder',
});

const result = await redactor.detect(text);
```

## Configuration Options

### Presets

OpenRedaction includes presets for compliance and industries, for example:

- **`gdpr`**, **`hipaa`**, **`ccpa`**
- **`pci-dss`**, **`soc2`**
- **`finance`**, **`education`**, **`healthcare`**, **`transportation`**

### Redaction modes

- **`placeholder`** — `[TYPE_n]` style tokens (default)
- **`mask-middle`**, **`mask-all`**, **`format-preserving`**, **`token-replace`**
